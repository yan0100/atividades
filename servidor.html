conceito de servidores

é um sistema computacional que processa solicitações e fornece serviços a outros computadores

tambem operam como o coração de uma rede de computadores, processando solicitações de dados e comandos

hardware

o hardware do servidores possuem peças semelhantes de um computador convencional, so que muito mais potentes.

como o procesador, a memoria ram, a placa mae, hd(disco rigido), fonte e refrigerador

placa mãe

a placa mãe é diferente do computador convencional, possui mais slotes para mais memoria ram e possui dois soquetes para o processador

processador

o processadores de servidores eles sao mais robustos para aguentar um grande fluxo de dados e possuem muito mais nucleos, velocidade, que os computadores pessoais

comparação, de um lado um processador grande que possui 24 nucleos para servidores e do outro um processador de 6 nucleos

memoria ram

a memoria ram do servidor suporta um grande volume de dados na memoria do servidor, estabilidade na memoria de servidor possui uma tecnologia ECC(Error Correction Code) ou codigo de correção de erro que detecta e corrige o erro em dados bom para trassmissao de dados

armazenamento

os servidores utilizam hd e ssd como armazenamento mas com verções com velocidade maiores de leitura e escrita para armazenar dados para executar serviços como rede em nuvem>

refrigeraçao

um servidor pode ser feita a ar ou liquido e é ajustada e toda ligação é feita canos para tentar remover todo o calor do servidor
